# Security Policy

## Supported Versions

Only the latest release version on the `main` branch is supported with security updates.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0.0 | No        |

## Reporting a Vulnerability

We take the security of this project seriously. If you find a security vulnerability, please do NOT open a public issue. Instead, report it privately to the maintainer:

- **Contact**: Divy Yadav
- **Email**: yadavdivy007@gmail.com

Please include:
- A description of the vulnerability.
- The potential impact.
- Steps, code, or a proof of concept (PoC) to reproduce the vulnerability.

We will acknowledge your report within 48 hours and work on a security fix. A public advisory will be drafted once the fix is deployed.
